# Prowler

[Prowler](https://prowler.com) is the world's most widely adopted open cloud security platform. It detects misconfigurations and vulnerabilities across AWS, Azure, GCP, Kubernetes, and more — prioritizes risk, accelerates remediation, and supports continuous compliance monitoring.

What started as a command-line AWS security assessment tool has grown into a full code-to-cloud platform, with hundreds of built-in checks covering CIS, PCI-DSS, NIST, GDPR, and other frameworks, plus support for custom controls.

## Connection to CCC

Prowler is part of the CCC ecosystem as an evaluation and detection layer. CCC catalogs define the controls and assessment requirements; Prowler runs those checks against live cloud environments and surfaces findings teams can act on — bridging the gap from open standards to automated cloud detections and remediations.
